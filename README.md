* BMI-Calculator: BMI 계산기

* Quiz-App: 퀴즈 게임
* Xylophone: 도레미파솔라시 음정을 내는 실로폰
* Tipsy: Tip을 포함한 총액을 N명끼리 나누면 인당 지불해야 하는 금액을 알려주는 계산기
* Clima: 검색한 도시의 현재 날씨 및 기온 확인
* ByteCoin: 현 시각 기준 비트코인의 가격을 여러 통화로 환산한 금액 확인
